## 📌 Project Overview

This project detects whether a news article is Fake or Real using Natural Language Processing and Machine Learning algorithms.

## 🎯 Objective

To identify misinformation and classify news content into:

* ✅ Real News
* ❌ Fake News

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* NLP
* TF-IDF Vectorization

## 🔄 Workflow
